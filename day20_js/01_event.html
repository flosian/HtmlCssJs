document 영역에서 사용되는 이벤트를 정의

  • 마우스, 키보드, 실행 등등 이벤트 종류가 매우 많기 때문에 웹에서 사용하는 이벤트를 기준으로 학습
  • 마우스 이벤트 : click(), focus(), input(), change(), mouseover(), mouseout(), scroll(), load() ...
  • 키보드 이벤트 : keyDown(), keyPress(), keyUp() ...
  • 기타 이벤트 관련된 메서드 : preventDefalut(), target, submit()
  • addEventListener('eventName', callbackFunction) 메서드 사용을 권장 같은 엘리먼트에 여러 이벤트 중복 부여 가능. 좀 더 편리한 형태.
